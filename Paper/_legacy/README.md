# Priced to Equivocate — LaTeX package

Research paper: **"Priced to Equivocate: Sub-Threshold Committee Bribery and the
Accountability Race in Sharded Blockchains."**

The formal model, threat model, reconfiguration analysis, propositions, and the
evaluation are all backed by a reproducible artifact (`../artifact/`). Every
figure and data table is generated from one configuration file; there are no
synthetic placeholders.

## Source layout (single source of truth)

The paper body lives in **`../Section/*.tex`** and is shared verbatim by all
three build wrappers. Edit the section files, not the wrappers or `body.tex`.

| File | Role |
|---|---|
| `../Section/*.tex` | The paper, section by section (abstract → conclusion). **Edit here.** |
| `../Section/_generated/` | Tables/macros generated by `../artifact/make_tables.py` (do not hand-edit). |
| `../artifact/figures/` | Figures generated by `../artifact/plot_figures.py`. |
| `paper.tex` | IEEEtran wrapper (Paper/ build). |
| `paper_article.tex` | `article` fallback wrapper (compiles without IEEEtran). |
| `body.tex` | Thin includer: `\input`s the `../Section/*.tex` files in order. |
| `common-preamble.tex` | Shared packages, theorem envs, macros, `\graphicspath`, listing style. |
| `refs.bib` | Bibliography (kept in sync with `../reference.bib`). |

The canonical IEEE build also exists at **`../ieee/main.tex`** (identical body,
its own preamble). All three resolve figures via `\graphicspath` and the
generated tables via `\input{../Section/_generated/...}`.

## Build

First (re)generate the artifact outputs the paper consumes, then compile.

```bash
# 1. artifact: data, figures, tables  (from ../artifact/)
python run_all.py --config configs/main.yaml
python plot_figures.py --input results/main.json --out figures/
python make_tables.py --input results/main.json --out ../Section/_generated

# 2a. canonical IEEE build (from ../ieee/)
pdflatex main && bibtex main && pdflatex main && pdflatex main

# 2b. or the Paper/ wrappers (from Paper/)
pdflatex paper && bibtex paper && pdflatex paper && pdflatex paper            # IEEEtran
pdflatex paper_article && bibtex paper_article && pdflatex paper_article x2   # article fallback
```

All three wrappers compile with no undefined references and no overfull boxes
(IEEEtran builds: 15 pp.; article fallback: 17 pp.).

## Status of the evaluation

The evaluation is produced by the artifact in `../artifact/`: a closed-form
model, Monte-Carlo simulators that validate it (the BFT harness reproduces the
closed-form success probability to within ~0.006), and a local end-to-end
proof-of-concept (an abstract BFT committee that produces real equivocation
evidence and drives a model `PayToEquivocate` escrow). No live network is
targeted; the PoC's signatures are toy and protect nothing.
